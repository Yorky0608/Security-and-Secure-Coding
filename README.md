# Security and Secure Coding

## RBAC Schedule App

This workspace contains a simple Python script that demonstrates:

- Username/password login
- One designated admin account
- Role-based access control (RBAC):
	- **Users** can view a schedule
	- **Admin** can change the schedule and create user accounts

### Demo accounts

- Admin: `admin / admin123`
- User: `user / password`

## CIA triad

The sort script implements Confidentiality because only the admin account is allowed to change the schedule.
